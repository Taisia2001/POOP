Paint commands

The File menu offers the following commands:

Open  Open an existing image.
Save   Save an opened image to a specified file name (format .png).
Exit  Exit Application.


The Edit menu offers the following commands:

Undo  Cancel an action.
Redo  Repeat a canceled action.
Cut  Cut a selected area and save it into the buffer.
Copy  Copy a selected area into buffer.
Paste  Paste an image from the buffer.


The Tools menu offers the following commands:

Color  Choose main color.
Fill Color  Choose color of rectangle and circle filling.
Width  Choose lines width.
Contour  Draw contours for rectangles and circles.
Filled  Fill rectangles and circles inside.
Font  Choose font for text.
Clear  Clear the canvas.


The Help menu offers the following commands:

Help  Show help dialog.
